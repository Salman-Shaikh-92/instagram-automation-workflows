# Instagram Automation Workflows

This repository contains automation workflows that automate Instagram posting and comment replies using automation tools.

The goal of this project is to demonstrate how social media tasks can be automated to save time and improve engagement.

---

## 1. Instagram Post Automation

This workflow automatically publishes Instagram posts or reels using data from Google Sheets.

### Workflow Process

1. A new row is added in Google Sheets
2. The automation reads the post content
3. Router determines the type of post (photo or reel)
4. The content is automatically published to Instagram
5. Google Sheets is updated with the post status

### Tools Used

- Google Sheets
- Instagram for Business
- Make Automation Platform

---

## 2. Instagram Auto Comment Reply

This workflow automatically replies to comments on Instagram posts.

### Workflow Process

1. A new comment appears on an Instagram post
2. The automation detects the comment
3. A predefined reply is triggered
4. The reply is automatically posted as a comment response

### Tools Used

- Instagram for Business
- Make Automation Platform
